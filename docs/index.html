How to make your own

Install the skill (one-line clone — see the README), then in Claude Code ask for something like:

"make me a cheatsheet for git rescue commands"

Claude will ask which specific CLIs to cover, capture real --help output for each, and write you both a markdown reference and an interactive HTML page like the ones above.
