Flags used

Public key algorithm to use (must precede -pkeyopt). RSA, EC, ED25519, ED448, X25519, X448, etc.

2. Generate an EC key

Elliptic-curve keys are dramatically smaller and faster than RSA at equivalent security. P-256 is the standard default; bump to P-384 or P-521 for higher security margins:

bash
Explain

Generates an EC private key on the P-256 curve and writes it to ec.key. Equivalent to secp256r1 / prime256v1.

Flags used

-algorithm EC
Select the elliptic-curve algorithm family.
-pkeyopt ec_paramgen_curve:NAME
Pick the named curve. Common values: P-256, P-384, P-521, secp256k1.
-out file
Output path for the PEM-encoded key.

Other flags you might want

-pkeyopt ec_param_enc:named_curve
Encode the curve by OID name (default and recommended) rather than embedding raw parameters.
-text
Also print the key in human-readable form.
-pass arg
Encrypt the output PEM with a passphrase from the given source.
-algorithm ED25519
Alternative modern algorithm — no -pkeyopt needed, smaller keys, fast.

List supported curves with openssl ecparam -list_curves.

Output file for the CSR (default stdout).

Add an extension to the request, e.g. -addext "subjectAltName=DNS:example.com,DNS:www.example.com". Repeatable.

Output format. DER is the binary X.690 encoding; PEM is its base64 + headers wrapper.
-out file
Output file (default stdout).

Other flags you might want

-inform DER | PEM | NET
Input format (default PEM). Use -inform der to convert a binary cert back to PEM.
-text
Print the human-readable form alongside the output.
-fingerprint
Quick way to confirm the converted file matches by hash.

Same pattern works for keys via openssl pkey -in key.pem -outform der -out key.der.

Flags used

-sha256
Use SHA-256 as the message digest.

Other flags you might want

-sha1 / -sha384 / -sha512 / -md5
Pick an alternative digest algorithm.
-hex
Output as hex (default for most digests).
-binary
Raw binary digest — pipe into xxd, base64, etc.
-r
coreutils-style output (<digest> <file>) — easy to feed back into sha256sum -c.

-out file
